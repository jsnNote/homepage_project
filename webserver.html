웹 서버란?

정의

클라이언트(일반적으로 웹 브라우저)의 요청에 따라 웹 사이트의 콘텐츠를 제공하는 서버이다.

특징

  • 항상 실행 중이다.
  • 항상 인터넷과 연결되어 있다.
  • 항상 같은 IP주소를 가지고 있다.
  • 제 3자에 의해 유지보수된다.
웹 서버의 구분
하드웨어 측면
웹 서버는 웹 서버 소프트웨어(apache http, Nginx)와 웹 사이트의 컴포넌트 파일들을 저장하는 물리적인 컴퓨터를 의미한다.
소프트웨어 측면
클라이언트로부터 HTTP 요청을 받아 요청을 분석하고 처리하여, 정적인 컨텐츠(.html, 이미지, CSS, JavaScrip 파일 등)를 제공하는 컴퓨터 프로그램을 의미한다. 사용자가 어떻게 호스트 파일들에 접근하는지를 관리한다.
웹 서버의 기능
1

    정적인 컨텐츠 제공

  • 웹 리소스들을 클라이언트로 전달하고, 클라이언트로부터 콘텐츠를 전달받아 저장하거나 처리한다.
  • WAS(웹 애플리케이션 서버; 동적인 컨텐츠를 처리하는 서버)를 거치지 않고 바로 자원을 제공한다.
2

    동적인 컨텐츠 제공

  • 웹 서버는 동적인 컨텐츠를 자체적으로 처리하기 어렵다.
  • 클라이언트의 요청을 WAS에 보내고, WAS가 처리한 결과를 클라이언트에게 전달(응답)한다.
